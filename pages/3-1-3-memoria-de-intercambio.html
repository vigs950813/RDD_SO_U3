3.1.3 Memoria de intercambio

Los sistemas operativos utilizan la memoria de intercambio, también conocida como swap, para administrar la memoria virtual. El sistema operativo selecciona los bloques de datos que son menos utilizados y los transfieren al almacenamiento secundario cuando la RAM está llena. Esto libera espacio en la RAM para otros procesos que lo necesiten. El sistema operativo recupera los datos en la RAM cuando se necesitan nuevamente. Si ocurre con demasiada frecuencia, este proceso de intercambio puede ser visible para el usuario, sus ventajas son:

  1. Permite ejecutar aplicaciones que requieren más memoria que la disponible físicamente.
  2. Ayuda a evitar que los programas se cierren inesperadamente debido a la falta de memoria.
  3. Facilita la multitarea al permitir que múltiples programas se ejecuten simultáneamente.

Desventajas

  1. Puede ralentizar el sistema si hay un intercambio excesivo de datos entre la RAM y el almacenamiento secundario.
  2. Aumenta la carga en el disco duro o la unidad de estado sólido, lo que puede reducir su vida útil.
